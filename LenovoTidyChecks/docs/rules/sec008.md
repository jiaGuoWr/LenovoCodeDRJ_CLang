# SEC008 - 不安全临时文件（Insecure Temp File）

| 属性 | 取值 |
|---|---|
| **ID** | `SEC008` / `lenovo-sec008-insecure-temp-file` |
| **类别** | Security |
| **严重级别** | Warning |
| **默认启用** | 是 |
| **对齐 C#** | [`SEC008` in AnalyzerRules.md](../../AnalyzerRules.md#sec008---insecure-temporary-file) |

## 描述

`tmpnam` / `mktemp` / `tempnam` / `tmpnam_r` 等 API 仅返回一个名字，**不原子地创建文件**，存在 TOCTOU（time-of-check vs time-of-use）漏洞。攻击者可在你打开文件之前抢占同名链接到敏感路径。

推荐改用：

- POSIX 上：`mkstemp` / `mkstemps` / `mkdtemp`（原子创建 + 文件描述符）
- C++17 起：`std::filesystem::temp_directory_path()` 配合 UUID/随机后缀 + `std::ofstream`(O_EXCL)

## 触发示例

```cpp
char buf[L_tmpnam];
std::tmpnam(buf);    // ❌
::mktemp(tpl);        // ❌
::tempnam(nullptr, "pre");  // ❌
```

## 合规示例

```cpp
char tpl[] = "/tmp/myappXXXXXX";
int fd = ::mkstemp(tpl);
```
