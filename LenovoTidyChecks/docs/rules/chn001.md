# CHN001 - 注释中的中文字符（Chinese in Comments）

| 属性 | 取值 |
|---|---|
| **ID** | `CHN001` / `lenovo-chn001-chinese-comments` |
| **类别** | Localization |
| **严重级别** | Warning |
| **默认启用** | 是 |
| **对齐 C#** | [`CHN001` in AnalyzerRules.md](../../AnalyzerRules.md#chn001---chinese-characters-in-comments) |

## 描述

为保证国际化团队协作，代码注释应使用英文。本规则遍历 `//` 单行与 `/* */` 块注释，一旦发现 CJK 字符即发出警告。

判定的 Unicode 范围：

- `U+3000 ~ U+303F` CJK Symbols & Punctuation
- `U+3400 ~ U+4DBF` CJK Unified Extension A
- `U+4E00 ~ U+9FFF` CJK Unified Ideographs（常用汉字）
- `U+F900 ~ U+FAFF` CJK Compatibility Ideographs
- `U+FF00 ~ U+FFEF` Halfwidth/Fullwidth Forms
- `U+20000 ~ U+2FFFF` CJK Unified Extension B–F

## 消息格式

```
comment contains Chinese characters [lenovo-chn001-chinese-comments]
```

## 触发示例

```cpp
// ❌ 触发 CHN001
// 这是一个中文注释

int x = 0; // 行尾中文注释

/* 块注释中的中文 */

/*
 * Multi-line comment
 * 混合英文 and 中文 content
 */
```

## 合规示例

```cpp
// ✅ Plain English comments
/* ASCII-only block comments */
/*
 * Multi-line English comment with numbers 123
 * and symbols !@#$%
 */
```

## 配置选项

当前版本无可调选项。未来版本计划加入：

| 选项（规划中） | 默认值 | 说明 |
|---|---|---|
| `AllowedRanges` | 空 | 允许的额外 Unicode 范围，例如仅放行日文片假名 |
| `IgnoreDoxygenTags` | `false` | 跳过 `@chinese_name` 这类 Doxygen 专用字段 |

## 抑制方式

```cpp
// NOLINT(lenovo-chn001-chinese-comments)
// 这行中文暂时保留
```

或在 `.clang-tidy` 中按目录/文件粒度禁用：

```yaml
# 允许 docs/ 下的 demo 文件保留中文注释
HeaderFilterRegex: '^(?!docs/).*'
```

## 已知限制

- 字符串字面量中的中文不会触发此规则（这是刻意设计；字符串里的中文常用于 UI 文案）
- 中文命名的宏或函数名不会触发（命名规则请参见 NAME001）
- `#include` 路径、行号宏等预处理产物不会触发
