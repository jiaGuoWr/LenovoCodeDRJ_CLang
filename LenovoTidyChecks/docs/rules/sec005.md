# SEC005 - 不安全随机数（Insecure Random）

| 属性 | 取值 |
|---|---|
| **ID** | `SEC005` / `lenovo-sec005-insecure-random` |
| **类别** | Security |
| **严重级别** | Warning |
| **默认启用** | 是 |
| **对齐 C#** | [`SEC005` in AnalyzerRules.md](../../AnalyzerRules.md#sec005---insecure-random-number-generator) |

## 描述

检测对 `rand` / `srand` / `rand_r` / `drand48` 系列伪随机数函数的直接调用。这些函数可预测，不应用于任何安全敏感场景（密钥、Token、CSRF Nonce 等）。

## 触发示例

```cpp
unsigned token = std::rand();   // ❌
std::srand(42);                  // ❌
```

## 合规示例

```cpp
#include <random>
std::random_device rd;
std::mt19937 eng(rd());
std::uniform_int_distribution<unsigned> dist;
unsigned token = dist(eng);

// 或使用 OpenSSL/CNG 等密码学库
```

## 已知限制

- 不区分"安全敏感场景 vs 仿真模拟"——后者其实可以用 `rand`，但不在自动判定能力内
- 可通过抑制注释或 `.clang-tidy` 排除来豁免
